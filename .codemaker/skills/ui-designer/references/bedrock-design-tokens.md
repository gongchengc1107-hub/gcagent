# Bedrock Design Tokens 速查表

> 绘制 UI 设计稿时，所有颜色、间距、字号必须严格使用以下规范，禁止硬编码任何色值。

---

## 颜色系统

### 文字颜色

| 用途 | CSS 变量 | Tailwind 类名 | 使用场景 |
|------|---------|--------------|---------|
| 主要文字 | `--text-color-1` | `text-color-1` | 标题、重要信息 |
| 次要文字 | `--text-color-2` | `text-color-2` | 正文、描述 |
| 辅助文字 | `--text-color-3` | `text-color-3` | 提示、注释 |
| 禁用文字 | `--text-color-4` | `text-color-4` | 禁用状态文字 |

### 背景颜色

| 用途 | CSS 变量 | Tailwind 类名 | 使用场景 |
|------|---------|--------------|---------|
| 页面背景 | `--bg-1` | `bg-1` | 整体页面底色 |
| 卡片背景 | `--bg-2` | `bg-2` | 卡片、面板背景 |
| 悬浮背景 | `--bg-3` | `bg-3` | 悬浮层、抽屉背景 |

### 填充颜色（交互状态）

| 用途 | CSS 变量 | Tailwind 类名 | 使用场景 |
|------|---------|--------------|---------|
| 轻度填充 | `--fill-1` | `bg-fill-1` | hover 背景、选中行 |
| 中度填充 | `--fill-2` | `bg-fill-2` | 标签背景 |
| 重度填充 | `--fill-3` | `bg-fill-3` | 禁用背景 |
| 强填充 | `--fill-4` | `bg-fill-4` | 分隔线、边框填充 |

### 主题色

| 用途 | CSS 变量 | Tailwind 类名 | 使用场景 |
|------|---------|--------------|---------|
| 主色（深） | `--primary-1` | `text-primary-1 / bg-primary-1` | 主按钮背景、链接 |
| 主色（中） | `--primary-2` | `text-primary-2 / bg-primary-2` | hover 主色 |
| 主色（浅） | `--primary-3` | `text-primary-3 / bg-primary-3` | 选中态背景 |
| 主色（更浅）| `--primary-4` | `text-primary-4 / bg-primary-4` | 轻度强调背景 |
| 主色（最浅）| `--primary-5` | `text-primary-5 / bg-primary-5` | 极浅强调背景 |

### 边框颜色

| 用途 | CSS 变量 | Tailwind 类名 | 使用场景 |
|------|---------|--------------|---------|
| 主边框 | `--border-color-1` | `border-color-1` | 卡片、输入框边框 |
| 次边框 | `--border-color-2` | `border-color-2` | 分隔线 |
| 轻边框 | `--border-color-3` | `border-color-3` | 表格行分隔 |

### 状态颜色

| 状态 | 色阶 | CSS 变量格式 | 使用场景 |
|------|------|------------|---------|
| 危险/错误 | 1~8 | `--status-danger-{N}` | 删除、拒绝、报错 |
| 成功 | 1~8 | `--status-success-{N}` | 通过、完成、正常 |
| 警告 | 1~8 | `--status-warning-{N}` | 待处理、注意 |
| 信息 | 1~8 | `--status-info-{N}` | 提示、进行中 |

**色阶使用规范**：
- `1` = 深色（文字/图标）
- `2~4` = 中色（标签背景、按钮）
- `5~8` = 浅色（浅色背景、hover 态）

### 阴影

| 用途 | CSS 变量 | Tailwind 类名 | 使用场景 |
|------|---------|--------------|---------|
| 轻阴影 | `--box-shadow-1` | `shadow-1` | 卡片悬浮 |
| 中阴影 | `--box-shadow-2` | `shadow-2` | 下拉菜单 |
| 重阴影 | `--box-shadow-3` | `shadow-3` | 弹窗 |
| 最重阴影 | `--box-shadow-4` | `shadow-4` | 抽屉 |

## 业务状态标签颜色规范

| 状态类型 | 推荐颜色变量 | 示例场景 |
|---------|-----------|---------|
| 待处理 / 待审核 | `--status-warning-1`（文）/ `--status-warning-5`（背）| 审核流程等待态 |
| 进行中 / 审核中 | `--status-info-1`（文）/ `--status-info-5`（背）| 流程进行态 |
| 已通过 / 已完成 | `--status-success-1`（文）/ `--status-success-5`（背）| 成功完结态 |
| 已拒绝 / 已驳回 | `--status-danger-1`（文）/ `--status-danger-5`（背）| 拒绝/失败态 |
| 已下线 / 已停用 | `--text-color-3`（文）/ `--fill-3`（背）| 停用/归档态 |
| 草稿 / 未提交 | `--text-color-2`（文）/ `--fill-2`（背）| 初始草稿态 |

## 间距系统（4px 栅格）

| Tailwind 类名 | 实际像素值 | 使用场景 |
|-------------|---------|---------|
| `p-px` / `gap-px` | 1px | 极细分隔线 |
| `p-px2` | 2px | 图标内边距 |
| `p-px6` | 6px | 小型标签内边距 |
| `p-1` / `gap-1` | 4px | 图标与文字间距 |
| `p-2` / `gap-2` | 8px | 行内元素间距 |
| `p-3` / `gap-3` | 12px | 小组件内边距 |
| `p-4` / `gap-4` | 16px | 标准内边距（卡片、表单） |
| `p-5` / `gap-5` | 20px | 模块间距 |
| `p-6` / `gap-6` | 24px | 大模块内边距 |
| `p-8` / `gap-8` | 32px | 页面内容区边距 |
| `p-10` / `gap-10` | 40px | 大区域间距 |
| `p-12` / `gap-12` | 48px | 页面顶部间距 |

## 字号系统

| Tailwind 类名 | font-size / line-height | 使用场景 |
|-------------|------------------------|---------|
| `text-sm` | 12px / 18px | 辅助说明、标签、注释 |
| `text-base` | 14px / 22px | 正文、表格内容（默认） |
| `text-lg` | 16px / 24px | 小标题、卡片标题 |
| `text-xl` | 18px / 26px | 中标题、模块标题 |
| `text-xxl` | 20px / 28px | 页面标题 |

## 圆角规范

| 场景 | 圆角值 | 说明 |
|------|-------|------|
| 小型标签、徽标 | 4px | Tag、Badge |
| 标准组件 | 6px | Button、Input、Select |
| 卡片、面板 | 8px | Card、Panel |
| 大型弹窗 | 12px | Modal、Drawer |
| 全圆角 | 9999px | 圆形按钮、头像 |

## 常用尺寸规范

### 组件高度

| 组件 | 高度 | 说明 |
|------|------|------|
| 顶部导航栏 | 56px | Header |
| 侧边栏 | 根据内容 | 宽度通常 200~240px |
| 标准按钮（大） | 40px | 主操作按钮 |
| 标准按钮（中） | 32px | 常规操作按钮 |
| 标准按钮（小） | 24px | 紧凑型按钮 |
| 输入框 | 32px | 标准输入高度 |
| 表格行高 | 48px | 标准数据行 |
| 表格行高（紧凑）| 40px | 数据密集型表格 |

### 弹窗尺寸

| 规格 | 宽度 | 使用场景 |
|------|------|---------|
| 小弹窗 | 400px | 确认弹窗、简单表单 |
| 中弹窗 | 560px | 标准表单弹窗 |
| 大弹窗 | 720px | 复杂表单、详情预览 |
| 超大弹窗 | 900px | 多步骤流程、大数据展示 |

> 弹窗高度不超过视口高度的 80%，超出时内容区滚动。
